SOPLang in Markdown

SOPLang (Standard Operating Procedure Language) can be embedded directly into Markdown files, allowing you to create rich, executable documentation.

To embed a SOPLang script, you can use a fenced code block with the soplang language identifier:

```soplang
# This is a SOPLang script
agent "MyAgent" do
  action "do_something"
end
```

Execution

When you open a Markdown file containing a SOPLang script in the Achilles IDE, you can execute it using the @soplangAgent.

The agent will parse the script and execute the defined procedures. The output of the execution will be displayed in the IDE's output panel.
