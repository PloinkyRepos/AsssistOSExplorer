Features & Screenshots

This page provides a visual overview of the key features of the Achilles IDE.

File & Folder Navigation

A clean, tree-based view of your workspace filesystem.

Markdown Document View

Rich rendering of .md files, parsed into chapters and paragraphs.

Contextual Plugin Toolbar

Plugin icons appear in the toolbar for the document, chapters, and paragraphs, providing contextual actions.

Plugins can open modals to interact with the user and perform actions.
