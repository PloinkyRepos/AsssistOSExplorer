What is the Explorer?

The Explorer is a multi-faceted agent that serves as the primary user interface and file management hub within the Ploinky workspace. It is a containerized Node.js application that provides a web-based UI and exposes a set of secure filesystem MCP (Model Context Protocol) tools.

High-Level Flow

The following diagram illustrates the high-level data flow of the Explorer agent:

User Browser (Web UI built on WebSkel)
    |
    | HTTP (Served by the Explorer container)
    v
Explorer Backend (filesystem-http-server.mjs)
    - Serves the static UI (HTML, JS, CSS)
    - Exposes MCP filesystem tools for other agents
    - Handles file uploads via the /blobs endpoint
    - Discovers plugins and serves their manifests to the UI
    |
    v
Workspace Filesystem (Mounted into the container)
