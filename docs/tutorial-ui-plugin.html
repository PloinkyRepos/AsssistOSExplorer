UI Plugin Tutorial

This tutorial will guide you through the process of creating a simple UI plugin for the Achilles IDE.

Step 1: Scaffolding your plugin

First, you need to create a new directory for your plugin. The directory structure should look like this:

/my-ui-plugin
|-- /ui
|   |-- index.html
|   |-- script.js
|   `-- style.css
`-- manifest.json

Step 2: The Plugin Manifest

The manifest.json file describes your plugin to the IDE. Here‚Äôs an example:

{
  "name": "My UI Plugin",
  "version": "1.0.0",
  "type": "ui",
  "entrypoint": "ui/index.html"
}

Step 3: The UI Code

Now, you can create the UI for your plugin. This is standard HTML, CSS, and JavaScript.

ui/index.html

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello from my UI Plugin!</h1>
  <script src="script.js"></script>
</body>
</html>

ui/style.css

body {
  font-family: sans-serif;
  text-align: center;
}

ui/script.js

console.log("My UI Plugin has loaded!");

Step 4: Building and Installing

To see your plugin in the IDE, you need to add it to the IDE's workspace and rebuild.

Instructions on how to build the workspace can be found on the Build page.
